# To---Do-List-Application-and-Remainders
A Python Tkinter-based To-Do List and Reminder app for managing daily tasks and alerts with simple data storage.
# Abstract
To-Do List Application is a Python-based GUI project developed using the Tkinter Library.It helps users organize and manage daily tasks efficiently.Users can add new tasks,remove completed tasks,and clear all tasks from the list.the application provides a simple  and interactive graphical interface,making task management easy and convenient.This project demonstrates the use of Python functions,lists,event handling,and GUI programming concepts.
## Features
- Add task
- Delete task
- Clear tasks
## Technologies used
- Python
- Tkinter
- Messagebox
